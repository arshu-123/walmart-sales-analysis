# Walmart Sales Performance Analysis

## Overview
Analyzed 1,000+ Walmart sales transactions using SQL for data cleaning, 
feature engineering, and business analysis.

## Key Findings
- Total Revenue: 3.23 lakh
- Top Branch: Naypyitaw (Branch C) — 1.10 lakh
- Top Product: Food and Beverages — 56,145
- Peak Month: January — 1.16 lakh
- Most Used Payment: Ewallet (345 transactions)

## Tools Used
- MySQL
- Power BI
- SQL

## Dataset
- 1,000 rows, 17 original columns → 20 after feature engineering
- 3 branches: Yangon, Mandalay, Naypyitaw
- Period: January – March 2019

## Feature Engineering
Added 3 new columns using SQL:
- time_of_day (Morning/Afternoon/Evening)
- day_name (DAYNAME function)
- month_name (MONTHNAME function)
